Адаптивный Barbershop Borodinski

Работа месяца

Проект представляет собой сайт мастерской, занимающейся исключительно мужскими стрижками. Страницы сверстаны адаптивно с резиновой сеткой, которая перестраивается на соответствующих ширинах вьюпортов с помощью медиавыражений:

  • мобильное уствойство: ширина до 767px;
  • планшет: ширина от 768px;
  • версия для рабочего стола: от 1200px.

В процессе были изучены:

  • адаптивность графики, чтобы загружать контентные изображения для мобильного, планшетного или десктопного разрешения на соответствующих ширинах вьюпортов;
  • графика для высокой плотности пикселей. Производилась подготовка растровых изображений для экранов с повышенной плотностью пикселей. Для наглядности на них добавлен полупрозрачный шрифт «х2»;
  • «нежная ретинизация». Загрузка изображений с повышенным разрешением происходит только на устройствах, с ретиновыми экранами, в то время, как устройства со стандартной плотность пикселей загружают обычные изображения. Это пример прогрессивного улучшения;
  • работа с препроцессорами, использование медиавыражений, примесей, переменных;
  • использование инструмента для автоматизации Gulp;
  • работа с Node.js, с помощью которого запускался сервер.

В рамках проекта реализовано:

  • именование классов и создание пространства имен согласно методологии БЭМ;
  • при стилизации наполнения сайта, использовался препроцессор LESS;
  • адаптивность графики;
  • графика для высокой плотности пикселей;
  • «нежная ретинизация»;
  • подключение нестандартного шрифта;
  • предусмотрено использование альтернативного «безопасного» шрифта;
  • использование спрайтов для ускорения загрузки страницы и уменьшения количества запросов на сервер;
  • для фоновых изображений предусмотрен альтернативный усредненный цвет;
  • минификация CSS и JS для более быстрой загрузки страницы;
  • слайдер переключения блоков с информацией на мобильной версии, реализованный средствами CSS;
  • стилизация стандартных input;
  • CSS анимация для оживления всплывающего окна авторизации;
  • предусмотрен отключенный JS для главной навигации в мобильной версии. В этом случае отображаться будут все пункты меню, а не только активный.

На каждой странице также фиксированно расположена ссылка, которая возвращает обратно на страницу портфолио.
